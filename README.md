# ops200-webpack-workshop

## Purpose of Program
Creates a simple web page to test webpack for a webpack tutorial.

## Setup

### Clone
Clone respository
### Install Dependencies
```
$ npm install
$ npm install -g webpack webpack-cli
```
### Run
1. Open a browser at localhost: 3000

2. Build and start server
```
$ npm run build
$ npm start
```
3. Testing
```
$ npm test
```
